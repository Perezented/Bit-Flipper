Arrow keys increment/decrement the numeric input
- ArrowUp / ArrowDown change value by 1
- Shift + Arrow -> change by 10
- Ctrl  + Arrow -> change by 100
- Alt   + Arrow -> change by 1000

Type a number to view its bits — bytes are 8-bit grids that light up green with a neon glow.
